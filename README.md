## Installation

1. Kloning repositori:

```bash
git clone git clone https://github.com/RafiNashirudin/API-Automation-Tests-Setup.git
cd api-automation-tests
```

2. Instal dependensi:

```bash
npm install
```

## jalankan Test

Untuk menjalankan tes, gunakan perintah berikut:

```bash
npm test
```
